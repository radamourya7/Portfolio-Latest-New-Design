# Professional CSE Portfolio 🚀

A modern, full-stack portfolio website built with the MERN stack (MongoDB, Express, React, Node.js). This project features a responsive design, smooth animations, and a functional contact system.

## 🌟 Key Features

- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Dynamic Animations**: Smooth transitions and interactive elements using `Framer Motion`.
- **Full-Stack Contact Form**: Integrated with Express and MongoDB to handle user inquiries.
- **Modern UI**: Clean and minimal design with a focus on usability.
- **Health Check API**: Built-in monitoring endpoint for system status.

## 🛠️ Tech Stack

### Frontend
- **React 18**: Component-based UI library.
- **Vite**: Ultra-fast build tool and development server.
- **Framer Motion**: For high-performance animations.
- **React Icons**: Comprehensive icon library.
- **React Scroll**: Smooth navigation between sections.

### Backend
- **Node.js**: JavaScript runtime environment.
- **Express**: Fast, unopinionated, minimalist web framework.
- **MongoDB**: NoSQL database for flexible data storage.
- **Mongoose**: Elegant mongodb object modeling for node.js.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (local or Atlas)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/radamourya7/Portfolio-Latest-New-Design.git
   cd Portfolio-Latest-New-Design
   ```

2. **Frontend Setup**
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. **Backend Setup**
   ```bash
   cd server
   npm install
   # Create a .env file based on the environment variables needed
   npm run dev
   ```

## 🌐 API Endpoints

- `GET /api/health`: Check if the server is running.
- `POST /api/contact`: Send a contact message.
